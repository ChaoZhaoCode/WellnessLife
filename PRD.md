# PRD.md — 长寿养生产品需求文档

> 本文档定义了"长寿养生"应用的产品需求、功能规划和设计方向。

---

## 1. 产品概述

### 1.1 产品定位

**长寿养生**是一个面向中老年人的健康知识科普应用。

核心理念：通过科普现代营养学、健康运动、睡眠和心态管理知识，帮助中老年人建立正确的养生观念，打破对补品的迷信。

产品口号：**"每天学一点，健康多一点"**

### 1.2 目标用户

| 维度 | 描述 |
|------|------|
| 年龄 | 55-75岁 |
| 性别 | 不限 |
| 地域 | 主要中文圈（中港台） |
| 特点 | 使用智能手机，依赖微信/短视频平台 |

**用户画像**：
- 李阿姨，62岁，退休教师
- 每天看微信养生文章和老姐妹分享的短视频
- 买过阿胶、燕窝、西洋参等多种补品
- 对"清淡饮食"、"喝粥养胃"深信不疑
- 需要大字、简单、好看的内容

### 1.3 核心需求

1. **知识获取** — 了解蛋白质、碳水、脂肪等基础营养概念
2. **辟谣打假** — 识别哪些补品是智商税
3. **简单易用** — 大字、清晰、好操作
4. **每日提醒** — 建立学习习惯

### 2.2 五大知识模块

#### 🍽️ 饮食板块（25张）

**基础篇（10张）**
- 蛋白质是什么/完全蛋白 vs 不完全蛋白
- 碳水化合物/升糖指数
- 好脂肪 vs 坏脂肪
- 膳食纤维
- 钙和维生素D

**实践篇（5张）**
- 老年人一天怎么吃
- 每顿饭蛋白质分配
- 买菜指南
- 食物搭配技巧

**辟谣篇（10张）**
- 阿胶/燕窝/骨头汤/西洋参/田七
- 清淡饮食误区/喝粥养胃/素食主义

## 7. 竞品分析

### 7.1 同类应用

| 应用 | 优点 | 不足 |
|------|------|------|
| 丁香医生 | 专业、内容丰富 | 太专业、不够通俗 |
| 养生堂 | 视频形式好 | 时长短、内容浅 |
| 微信养生文章 | 传播广 | 谣言多、不系统 |

### 7.2 差异化

1. **大白话 + 数据** — 既有科普深度，又有数据支撑
2. **辟谣特色** — 专门针对补品和传统误区
3. **卡片形式** — 适合碎片化学习
4. **成就激励** — 建立学习习惯
