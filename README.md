# 长寿养生

面向中老年人的健康知识科普应用。

## 核心理念

**"每天学一点，健康多一点"**

通过大白话科普 + 数据支撑辟谣，帮助中老年人建立正确的养生观念。

## 目标用户

- 年龄：55-75岁
- 特点：迷信补品，信息来自老姐妹推荐和短视频
- 核心需求：大字、简单、好看的内容

## 功能

- 📚 100张知识卡片
- 🍽️ 饮食 / 🧘 运动 / 😴 睡眠 / 🌸 心态 / 🚫 辟谣 五大模块
- 📖 分类浏览
- ⭐ 收藏功能
- 🏆 成就系统

## 快速开始

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build
```

## 访问地址

http://localhost:3000/zh/health

## 技术栈

- Next.js 14 + TypeScript
- Tailwind CSS
- shadcn/ui
- next-intl

## 项目结构

```
src/
├── app/[locale]/health/   # 养生知识页面
├── components/icons/       # SVG 图标
├── content/cards/          # 知识卡片内容
└── styles/                # 样式文件
```

## 内容分类

| 板块 | 数量 | 内容 |
|------|------|------|
| 饮食 | 25张 | 蛋白质/碳水/脂肪/补品辟谣 |
| 运动 | 25张 | 力量/关节/体态/防跌倒 |
| 睡眠 | 20张 | 基础/问题/习惯 |
| 心态 | 15张 | 情绪/生活/意义 |
| 辟谣 | 15张 | 补品/观念/中药 |

**总计：100张**
